LLogos is a native inference engine that runs quantized AI language models (in GGUF format) across every major compute platform. Written entirely in C# and TypeScript, it delivers hardware-direct performance from your gaming PC, a server cluster, or a web browser — with no Python dependencies and no wrapper libraries in the critical path.

## Why LLogos

LLogos targets developers who need inference performance without the Python ecosystem overhead. Every backend calls hardware APIs directly — CUDA Driver API via P/Invoke, Vulkan compute via SPIR-V shaders, WebGPU via WGSL — so there are no intermediate abstraction layers consuming cycles.
